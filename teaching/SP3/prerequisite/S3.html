Random variables

3 Random variables

Let 𝒮{\mathcal{S}} be a set. We say XX is a 𝒮{\mathcal{S}}-valued random variable on a probability space (Ω,ℱ,ℙ)(\Omega,{\mathcal{F}},\mathbb{P}) if it is a “nice” function X:Ω→𝒮X:\Omega\to{\mathcal{S}}. We are typically in one of the following scenarios:

  • •
    ​

    𝒮=ℝ{\mathcal{S}}=\mathbb{R}: in which case we say XX is a real-valued random variable.

  • •
    ​

    𝒮=ℝd{\mathcal{S}}=\mathbb{R}^{d}: in which case we often say XX is a (dd-dimensional real) random vector, since we can write X=(X1,…,Xd)X=(X_{1},\dots,X_{d}) where each of Xi:Ω→ℝX_{i}:\Omega\to\mathbb{R} is a random variable,

  • •
    ​

    X⁢(Ω)X(\Omega) (or 𝒮{\mathcal{S}}) is countable: in which case we say XX is a discrete random variable.

3.1 “Nice-ness”: measurable functions

Those of you who have done Probability II would have seen the proper definition of “nice-ness”, i.e. measurability, at least for real-valued random variables. This concept was also briefly mentioned in Probability I but was not examinable.

Let us now briefly discuss this concept with some examples/exercise. We will also go through this concept during the first few weeks of our lectures in Michaelmas when we review basic probability theory, but you are strongly encouraged to go through the explanations below before the start of the term.

Real-valued random variables.

In Probability II, the definition of a random variable probably goes like this: a (real-valued) random variable XX on a probablity space (Ω,ℱ,ℙ)(\Omega,{\mathcal{F}},\mathbb{P}) is a function X:Ω→ℝX:\Omega\to\mathbb{R} such that

X−1⁢((−∞,x]):={ω∈Ω:X⁢(ω)≤x}∈ℱ∀x∈ℝ.\displaystyle X^{-1}((-\infty,x]):=\{\omega\in\Omega:X(\omega)\leq x\}\in{% \mathcal{F}}\qquad\forall x\in\mathbb{R}. (3.1)

This definition may not look very motivated, and so we will consider the following definition instead and see why this is equivalent.

Definition 3.1.

We say XX is a real-valued random variable XX on a probability space (Ω,ℱ,ℙ)(\Omega,{\mathcal{F}},\mathbb{P}) if X:Ω→ℝX:\Omega\to\mathbb{R} is a function such that

X−1⁢(O):={ω∈Ω:X⁢(ω)∈O}∈ℱfor all open subsets O⊆Ω.\displaystyle X^{-1}(O):=\{\omega\in\Omega:X(\omega)\in O\}\in{\mathcal{F}}% \qquad\text{for all open subsets $O\subseteq\Omega$}. (3.2)
Exercise 3.2.

Show that the two definitions of real-valued random variable are equivalent.

Solution:

  • •
    ​

    Let us first show that (3.2) implies (3.1).

    Suppose XX is a random variable under the definition (3.2). We want to establish X−1⁢((−∞,x])∈ℱX^{-1}((-\infty,x])\in{\mathcal{F}}, but

    X−1⁢(⋂n=1∞(−∞,x+n−1))=⋂n=1∞X−1⁢((−∞,x+n−1))∈ℱ\displaystyle X^{-1}\left(\bigcap_{n=1}^{\infty}(-\infty,x+n^{-1})\right)=% \bigcap_{n=1}^{\infty}X^{-1}\left((-\infty,x+n^{-1})\right)\in{\mathcal{F}}

    which follows from axiom (SA-4) of σ\sigma-algebra (which followed from De Morgan’s law and the equivalent (SA-3) axiom).

  • •
    ​

    Let us show (3.1) implies (3.2). First, we show that X−1⁢((a,b])∈ℱX^{-1}((a,b])\in{\mathcal{F}} for any a<ba<b. This is true because

    X−1⁢((a,b])\displaystyle X^{-1}\left((a,b]\right) =X−1⁢((−∞,b])∖X−1⁢((−∞,a])\displaystyle=X^{-1}\left((-\infty,b]\right)\setminus X^{-1}\left((-\infty,a]\right)
    =X−1⁢((−∞,b])∩X−1⁢((−∞,a])c\displaystyle=X^{-1}\left((-\infty,b]\right)\cap X^{-1}\left((-\infty,a]\right% )^{c}

    and we just need to recall the fact that X−1⁢((−∞,a])c∈ℱX^{-1}\left((-\infty,a]\right)^{c}\in{\mathcal{F}} by (SA-2), and hence X−1⁢((−∞,b])∩X−1⁢((−∞,a])c∈ℱX^{-1}\left((-\infty,b]\right)\cap X^{-1}\left((-\infty,a]\right)^{c}\in{% \mathcal{F}} by (SA-3’).

    Next, we need to use a fact about open sets in ℝ\mathbb{R}: any open sets O⊆ℝO\subseteq\mathbb{R} can be written as a countable union of open intervals O=⋃n=1∞(an,bn)O=\bigcup_{n=1}^{\infty}(a_{n},b_{n}). Then

    X−1⁢(O)=⋃n=1∞X−1⁢((an,bn))\displaystyle X^{-1}(O)=\bigcup_{n=1}^{\infty}X^{-1}((a_{n},b_{n}))

    which is in ℱ{\mathcal{F}} if we know X−1⁢((an,bn))∈ℱX^{-1}((a_{n},b_{n}))\in{\mathcal{F}} for every n∈ℕn\in\mathbb{N} because of axiom (SA-3) of σ\sigma-algebras. But each (an,bn)(a_{n},b_{n}) can again be written as a countable union e.g. ∪k=1∞(an,bn−1k]\cup_{k=1}^{\infty}(a_{n},b_{n}-\frac{1}{k}] and so the claim follows from another application of (SA-3).

General 𝒮{\mathcal{S}}-valued random variables.

In order to talk about (Borel) random variables taking values in general set 𝒮{\mathcal{S}}, we need to equip 𝒮{\mathcal{S}} with a topology, i.e. which subsets O⊆𝒮O\subseteq{\mathcal{S}} are defined as open sets of 𝒮{\mathcal{S}}. Then:

Definition 3.3.

An 𝒮{\mathcal{S}}-valued random variable XX on a probability space (Ω,ℱ,ℙ)(\Omega,{\mathcal{F}},\mathbb{P}) is a function X:Ω→𝒮X:\Omega\to{\mathcal{S}} such that

X−1⁢(O):={ω∈Ω:X⁢(ω)∈O}∈ℱfor all open subsets O⊆𝒮.\displaystyle X^{-1}(O):=\{\omega\in\Omega:X(\omega)\in O\}\in{\mathcal{F}}% \qquad\text{for all open subsets $O\subseteq{\mathcal{S}}$}. (3.3)

One way to understand the definition of general random variables is to view it as generalisation of the concept of continuity: we say f:Ω→𝒮f:\Omega\to{\mathcal{S}} is a continuous function if

f−1⁢(O):={ω∈Ω:f⁢(ω)∈O}is an open subset of Ω.\displaystyle f^{-1}(O):=\{\omega\in\Omega:f(\omega)\in O\}\quad\text{is an % open subset of $\Omega.$} (3.4)

Discrete random variables.

When 𝒮{\mathcal{S}} is countable, we equip it with the discrete topology, i.e. all subsets of 𝒮{\mathcal{S}} (and in particular all singleton sets) are declared open in 𝒮{\mathcal{S}}. Under such scenario, X:Ω→𝒮X:\Omega\to{\mathcal{S}} is a random variable if and only if X−1⁢(s)={ω∈Ω:X⁢(ω)=s}∈ℱX^{-1}(s)=\{\omega\in\Omega:X(\omega)=s\}\in{\mathcal{F}} for all s∈𝒮s\in{\mathcal{S}}.

Note that this definition is compatible with the situation where 𝒮=ℝ{\mathcal{S}}=\mathbb{R} but X⁢(Ω)X(\Omega) is countable:

3.2 Independence

We say two random variables X1,X2X_{1},X_{2} are independent of each other if

ℙ⁢(X1∈A1,X2∈A2)=ℙ⁢(X1∈A1)⁢ℙ⁢(X2∈A2)\displaystyle\mathbb{P}(X_{1}\in A_{1},X_{2}\in A_{2})=\mathbb{P}(X_{1}\in A_{% 1})\mathbb{P}(X_{2}\in A_{2}) (3.5)

for all “nice” subsets A1A_{1} and A2A_{2}. When X1X_{1} and X2X_{2} are both discrete, (3.5) can be reduced to

ℙ⁢(X1=x2,X2=x2)=ℙ⁢(X1=x1)⁢ℙ⁢(X2=x2)∀(x1,x2)∈𝒳1×𝒳2.\displaystyle\mathbb{P}(X_{1}=x_{2},X_{2}=x_{2})=\mathbb{P}(X_{1}=x_{1})% \mathbb{P}(X_{2}=x_{2})\qquad\forall(x_{1},x_{2})\in{\mathcal{X}}_{1}\times{% \mathcal{X}}_{2}. (3.6)
Exercise 3.6.

Show the equivalence of (3.5) and (3.6) when the random variables X1,X2X_{1},X_{2} are discrete.

Solution: it is obvious that (3.5) is stronger than (3.6) (since we can always take A1={x1}A_{1}=\{x_{1}\} and A2={x2}A_{2}=\{x_{2}\}), so we only need to show that (3.6) implies (3.5) in the discrete case, but

ℙ⁢(X1∈A1,X2∈A2)\displaystyle\mathbb{P}(X_{1}\in A_{1},X_{2}\in A_{2}) =∑x1∈A1,x2∈A2ℙ⁢(X1=x1,X2=x2)\displaystyle=\sum_{x_{1}\in A_{1},x_{2}\in A_{2}}\mathbb{P}(X_{1}=x_{1},X_{2}% =x_{2})
=∑x1∈A1,x2∈A2ℙ⁢(X1=x1)⁢ℙ⁢(X2=x2)=ℙ⁢(X1∈A1)⁢ℙ⁢(X2∈A2)\displaystyle=\sum_{x_{1}\in A_{1},x_{2}\in A_{2}}\mathbb{P}(X_{1}=x_{1})% \mathbb{P}(X_{2}=x_{2})=\mathbb{P}(X_{1}\in A_{1})\mathbb{P}(X_{2}\in A_{2})

where the first equality follows from countable addivitiy of probability measure, and the second equality from independence in the sense of (3.6).

Observe that the notion of independence of random variables generalises that of independent events: E1,E2∈ℱE_{1},E_{2}\in{\mathcal{F}} are said to be independent if ℙ⁢(E1∩E2)=ℙ⁢(E1)⁢ℙ⁢(E2)\mathbb{P}(E_{1}\cap E_{2})=\mathbb{P}(E_{1})\mathbb{P}(E_{2}).

Exercise 3.8.

Mutual independence.

Let {Xi,i∈ℐ}\{X_{i},i\in{\mathcal{I}}\} be a collection of random variables (on the same probability space (Ω,ℱ,ℙ)(\Omega,{\mathcal{F}},\mathbb{P})). This collection may be finite, countably infinite, or even uncountably infinite. We say the random variables XiX_{i} are mutually independent if any finite subcollection is mutually independent, i.e. for any n∈ℕn\in\mathbb{N}, and any distinct i1,…,in∈ℐi_{1},\dots,i_{n}\in{\mathcal{I}}, we have

ℙ⁢(Xi1∈Ai1,…,Xin∈Ain)=ℙ⁢(Xi1∈Ai1)⁢⋯⁢ℙ⁢(Xin∈Ain)\displaystyle\mathbb{P}\left(X_{i_{1}}\in A_{i_{1}},\dots,X_{i_{n}}\in A_{i_{n% }}\right)=\mathbb{P}(X_{i_{1}}\in A_{i_{1}})\cdots\mathbb{P}(X_{i_{n}}\in A_{i% _{n}})

for any “nice” subsets Ai1,…,AinA_{i_{1}},\dots,A_{i_{n}}.

3.3 Distributions

ℙX:A↦ℙX(A):=ℙ(X∈A)for all “nice” subsets A⊆𝒮.\displaystyle\mathbb{P}_{X}:A\mapsto\mathbb{P}_{X}(A):=\mathbb{P}(X\in A)% \qquad\text{for all ``nice" subsets $A\subseteq{\mathcal{S}}$}.

When 𝒮=ℝd{\mathcal{S}}=\mathbb{R}^{d} and X=(X1,…,Xd)X=(X_{1},\dots,X_{d}), we also define the (joint) cumulative distribution function (c.d.f.) of XX as

FX⁢(x)=FX1,…,Xd⁢(x1,…,xd):=ℙ⁢(X1≤x1,…,Xd≤xd)∀x=(x1,…,xd)∈ℝd.\displaystyle F_{X}(x)=F_{X_{1},\dots,X_{d}}(x_{1},\dots,x_{d}):=\mathbb{P}% \left(X_{1}\leq x_{1},\dots,X_{d}\leq x_{d}\right)\qquad\forall x=(x_{1},\dots% ,x_{d})\in\mathbb{R}^{d}. (3.7)

ℙ⁢(F←X⁢(U)≤u)=ℙ⁢(U≤FX⁢(u))=∫0FX⁢(u)𝑑s=FX⁢(u)\displaystyle\mathbb{P}\left(\overleftarrow{F}_{X}(U)\leq u\right)=\mathbb{P}(% U\leq F_{X}(u))=\int_{0}^{F_{X}(u)}ds=F_{X}(u)

by the definition of F←X\overleftarrow{F}_{X} (and the fact that the c.d.f. FXF_{X} is right-continuous).

    ​

  • (iii)
    ​

  • (i)
    ​

    ​

    for any y≥0y\geq 0, i.e. YY is distributed according to Exp⁢(4)\mathrm{Exp}(4).
