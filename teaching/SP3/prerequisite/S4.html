4.1 Real-valued random variables

Let us recall the definition of expectation or expected value from first year probability: let XX be a real-valued random variable and g:ℝ→ℝg:\mathbb{R}\to\mathbb{R} is some “nice” function, then

𝔼⁢[g⁢(X)]:={∑x∈𝒳g⁢(x)⁢pX⁢(x)if 𝒳:={X⁢(ω):ω∈Ω} is discrete/countable∫ℝg⁢(x)⁢fX⁢(x)⁢𝑑xif X is absolutely continuous with p.d.f. fX⁢(⋅)\displaystyle\mathbb{E}\left[g(X)\right]:=\begin{dcases}\sum_{x\in{\mathcal{X}% }}g(x)p_{X}(x)&\text{if ${\mathcal{X}}:=\{X(\omega):\omega\in\Omega\}$ is % discrete/countable}\\ \int_{\mathbb{R}}g(x)f_{X}(x)dx&\text{if $X$ is absolutely continuous with p.d% .f. $f_{X}(\cdot)$}\end{dcases} (4.1)

provided that the sum/integral is well-defined, i.e. 𝔼⁢[|g⁢(X)|]<∞\mathbb{E}[|g(X)|]<\infty, in which case we say g⁢(X)g(X) is integrable.

    ​

    the variance of XX is defined as Var⁢(X):=𝔼⁢[(X−𝔼⁢[X])2]\mathrm{Var}(X):=\mathbb{E}[(X-\mathbb{E}[X])^{2}].

𝔼⁢[g⁢(X1,…,Xd)]\displaystyle\mathbb{E}[g(X_{1},\dots,X_{d})]

provided that the sum/integral is well-defined (i.e. integrable).

Given two real-valued random variables X1X_{1} and X2X_{2}, the covariance Cov⁢(X1,X2)\mathrm{Cov}(X_{1},X_{2}) is defined as 𝔼[(X1−𝔼[X1])(X2−𝔼[X2])\mathbb{E}[(X_{1}-\mathbb{E}[X_{1}])(X_{2}-\mathbb{E}[X_{2}]). Notice that this generalises the concept of variance, since Var⁢(X)=Cov⁢(X,X)\mathrm{Var}(X)=\mathrm{Cov}(X,X).

4.2 Properties of expectation

Given the definition of expected value (4.2) as a summation/integration, it is not hard to see that 𝔼⁢[⋅]\mathbb{E}[\cdot] is a linear map: if X1,…,XdX_{1},\dots,X_{d} are a collection of real-valued random variables and a1,…,ad∈ℝa_{1},\dots,a_{d}\in\mathbb{R} are constants, then

𝔼⁢[∑k=1dak⁢Xk]=∑k=1dak⁢𝔼⁢[Xk]\displaystyle\mathbb{E}\left[\sum_{k=1}^{d}a_{k}X_{k}\right]=\sum_{k=1}^{d}a_{% k}\mathbb{E}[X_{k}] (4.3)

provided that all the expectations appearing in the equality exist. This may be extended to a countable collection of random variables if certain conditions are satisfied (using monotone/dominated convergence theorems).

Another important property of expectation is factorisation when working with independent random variables. More precisely, if X1,X2X_{1},X_{2} are two independent (real-valued) random variables, then for any “nice” functions gi:ℝ→ℝg_{i}:\mathbb{R}\to\mathbb{R},

𝔼⁢[g1⁢(X1)⁢g2⁢(X2)]=𝔼⁢[g1⁢(X1)]⁢𝔼⁢[g2⁢(X2)]\displaystyle\mathbb{E}[g_{1}(X_{1})g_{2}(X_{2})]=\mathbb{E}[g_{1}(X_{1})]% \mathbb{E}[g_{2}(X_{2})]

provided that the expected values appearing on both sides on the equality are well-defined.

4.3 Elementary conditional expectations

Let XX be an integrable random variable and B∈ℱB\in{\mathcal{F}} be an event with strictly positive probability ℙ⁢(B)>0\mathbb{P}(B)>0. We define the conditional expectation of XX given BB as

𝔼⁢[X|B]:=𝔼⁢[X⁢1B]ℙ⁢(B).\displaystyle\mathbb{E}[X|B]:=\frac{\mathbb{E}\left[X1_{B}\right]}{\mathbb{P}(% B)}.

In particular, if A∈ℱA\in{\mathcal{F}} and X=1AX=1_{A}, then

𝔼⁢[X|B]=𝔼⁢[1A⁢1B]ℙ⁢(B)=ℙ⁢(A∩B)ℙ⁢(B)=ℙ⁢(A|B).\displaystyle\mathbb{E}[X|B]=\frac{\mathbb{E}[1_{A}1_{B}]}{\mathbb{P}(B)}=% \frac{\mathbb{P}(A\cap B)}{\mathbb{P}(B)}=\mathbb{P}(A|B).

Partition theorem.

More generally, if Π:={Bi∈ℐ}\Pi:=\{B_{i}\in{\mathcal{I}}\} is a partition of the sample space Ω\Omega with at most countably many parts BiB_{i}, then the partition theorem says

𝔼⁢[X]=∑i∈ℐ𝔼⁢[X|Bi]⁢ℙ⁢(Bi)\displaystyle\mathbb{E}[X]=\sum_{i\in{\mathcal{I}}}\mathbb{E}[X|B_{i}]\mathbb{% P}(B_{i})

provided that we interpret 𝔼⁢[X|Bi]⁢ℙ⁢(Bi)=0\mathbb{E}[X|B_{i}]\mathbb{P}(B_{i})=0 as soon as ℙ⁢(Bi)=0\mathbb{P}(B_{i})=0. This generalises the partition theorem for probability: if we take X=1AX=1_{A} for some event A∈ℱA\in{\mathcal{F}}, then XX is obviously integrable and

ℙ⁢(A)=𝔼⁢[X]=∑i∈ℐ𝔼⁢[1A|Bi]⁢ℙ⁢(Bi)=∑i∈ℐℙ⁢(A|Bi)⁢ℙ⁢(Bi).\displaystyle\mathbb{P}(A)=\mathbb{E}[X]=\sum_{i\in{\mathcal{I}}}\mathbb{E}[1_% {A}|B_{i}]\mathbb{P}(B_{i})=\sum_{i\in{\mathcal{I}}}\mathbb{P}(A|B_{i})\mathbb% {P}(B_{i}).

4.4 Generating functions

Let XX be a real-valued random variable.

  • •
    ​

    The moment generating function (m.g.f.) MX⁢(⋅)M_{X}(\cdot) is defined as MX⁢(t):=𝔼⁢[et⁢X]M_{X}(t):=\mathbb{E}[e^{tX}] for values of t∈ℝt\in\mathbb{R} such that et⁢Xe^{tX} is integrable.

  • •
    ​

    When XX only takes values in {0,1,…,}=:ℕ0\{0,1,\dots,\}=:\mathbb{N}_{0}, we also speak of the probability generating function

    GX⁢(t):=𝔼⁢[tX]=∑n=0∞pX⁢(n)⁢tn\displaystyle G_{X}(t):=\mathbb{E}[t^{X}]=\sum_{n=0}^{\infty}p_{X}(n)t^{n}

    for all values of t∈ℝt\in\mathbb{R} such that the above sum converges. Note that probability generating function always exists for |t|<1|t|<1 for such a random variable XX.

Multivariate generating functions.

Given a collection of real-valued random variables (X1,…,Xd)(X_{1},\dots,X_{d}), one can define analogous objects known as joint m.g.f.s and joint p.g.f.s, e.g.

MX1,…,Xd⁢(t1,…,td)\displaystyle M_{X_{1},\dots,X_{d}}(t_{1},\dots,t_{d}) :=𝔼⁢[exp⁡(∑k=1dtk⁢Xk)]\displaystyle:=\mathbb{E}\left[\exp\left(\sum_{k=1}^{d}t_{k}X_{k}\right)\right]
andGX1,…,Xd⁢(t1,…,td)\displaystyle\qquad\text{and}\qquad G_{X_{1},\dots,X_{d}}(t_{1},\dots,t_{d}) :=𝔼⁢[∏k=1dtkXk]\displaystyle:=\mathbb{E}\left[\prod_{k=1}^{d}t_{k}^{X_{k}}\right]

and similar properties hold but we do not discuss them further in this document.
