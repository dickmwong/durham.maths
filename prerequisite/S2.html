Probability space

2 Probability space

When we speak of a probability space, we mean a triple (Ω,ℱ,ℙ)(\Omega,{\mathcal{F}},\mathbb{P}):

2.1 Sample space

The (non-empty) set Ω\Omega is known as the sample space. This is the set of all possible outcomes in an experiment. We say a sample space is discrete when it is countable (finite or not).

Example 2.1.

In an experiment involving 55 coin tosses, the sample space would be given by

Ω:={ω=(ω1,…,ω5)∈{H,T}5},\displaystyle\Omega:=\{\omega=(\omega_{1},\dots,\omega_{5})\in\{\mathrm{H},% \mathrm{T}\}^{5}\},

i.e. each sample is of the form ω=(ω1,…,ω5)\omega=(\omega_{1},\dots,\omega_{5}) where ωi∈{H,T}\omega_{i}\in\{\mathrm{H},\mathrm{T}\} is the result in the ii-th coin toss.

Exercise 2.2.

What would be a sample space for an experiment with two dice throw?

Solution: Ω:=[6]×[6]={(i,j):i,j∈{1,…,6}}\Omega:=[6]\times[6]=\{(i,j):i,j\in\{1,\dots,6\}\}.

2.2 σ\sigma-algebra

ℱ{\mathcal{F}} is a σ\sigma-algebra on Ω\Omega, i.e. a collection of events under consideration that satisfies the following compatibility conditions:

  • (SA-1)
    ​

    ∅︀∈ℱ\emptyset\in{\mathcal{F}}.

  • (SA-2)
    ​

    If A∈ℱA\in{\mathcal{F}}, then Ac:=Ω∖A∈ℱA^{c}:=\Omega\setminus A\in{\mathcal{F}}.

  • (SA-3)
    ​

    If An∈ℱA_{n}\in{\mathcal{F}} for all n∈ℕn\in\mathbb{N}, then ⋃n=1∞An∈ℱ\bigcup_{n=1}^{\infty}A_{n}\in{\mathcal{F}}.

Exercise 2.3.

Show that (SA-3) can be replaced by

  • (SA-4)
    ​

    If An∈ℱA_{n}\in{\mathcal{F}} for all n∈ℕn\in\mathbb{N}, then ⋂n=1∞An∈ℱ\bigcap_{n=1}^{\infty}A_{n}\in{\mathcal{F}}.

Solution: this follows from De Morgan’s law: ⋂n=1∞An=(⋃n=1∞Anc)∈ℱ.\bigcap_{n=1}^{\infty}A_{n}=\left(\bigcup_{n=1}^{\infty}A_{n}^{c}\right)\in{% \mathcal{F}}.

When the underlying sample space Ω\Omega is discrete, we usually set ℱ=2Ω={\mathcal{F}}=2^{\Omega}= the collection of all subsets of Ω\Omega (including itself and the emptyset ∅︀\emptyset).

Exercise 2.4.

Let Ω={H,T}\Omega=\{\mathrm{H},\mathrm{T}\}. What is 2Ω2^{\Omega}?

Solution: 2Ω={∅︀,Ω,{H},{T}}2^{\Omega}=\{\emptyset,\Omega,\{\mathrm{H}\},\{\mathrm{T}\}\}.

Exercise 2.7.

Consider Ω=ℝ\Omega=\mathbb{R}. Let ℱ{\mathcal{F}} be the Borel σ\sigma-algebra of ℝ\mathbb{R}, i.e. the smallest σ\sigma-algebra containing all open sets of ℝ\mathbb{R}, and 𝒢{\mathcal{G}} be the smallest σ\sigma-algebra containing all singletons {x}\{x\}. Show that (i) 𝒢⊆ℱ{\mathcal{G}}\subseteq{\mathcal{F}} and (ii) ℱ⊈𝒢{\mathcal{F}}\not\subseteq{\mathcal{G}}.

Solution:

  • (i)
    ​

    We want to show that {x}∈ℱ\{x\}\in{\mathcal{F}} for each x∈ℝx\in\mathbb{R}. But then {x}c=(−∞,x)∪(x,∞)\{x\}^{c}=(-\infty,x)\cup(x,\infty) which is an open set that is contained in ℱ{\mathcal{F}} by definition. Using (SA-2), it follows that {x}=({x}c)c∈ℱ\{x\}=\left(\{x\}^{c}\right)^{c}\in{\mathcal{F}}.

  • (ii)
    ​

    Let us consider 𝒢′:={A⊆Ω:only one of A,Ac is uncountable}{\mathcal{G}}^{\prime}:=\{A\subseteq\Omega:\text{only one of $A,A^{c}$ is % uncountable}\}, which is a σ\sigma-algebra (check that (SA-1)-(SA-3) are indeed satisfied). Our claim is that 𝒢=𝒢′{\mathcal{G}}={\mathcal{G}}^{\prime}, from which we can deduce the result because e.g. (0,1),(0,1)c∈ℱ(0,1),(0,1)^{c}\in{\mathcal{F}} but both of these sets are uncountable, i.e. (0,1)∉𝒢(0,1)\not\in{\mathcal{G}}.

    To verify the claim, first observe that 𝒢⊆𝒢′{\mathcal{G}}\subseteq{\mathcal{G}}^{\prime}, since all singleton sets are automaticaly contained in 𝒢′{\mathcal{G}}^{\prime} by definition. On the other hand, if A∈𝒢′A\in{\mathcal{G}}^{\prime} is countable, then A=⋃x∈A{x}∈𝒢A=\bigcup_{x\in A}\{x\}\in{\mathcal{G}} by (SA-3); if A∈𝒢′A\in{\mathcal{G}}^{\prime} then we instead establish Ac∈𝒢A^{c}\in{\mathcal{G}} using the same argument, from which we deduce A∈𝒢A\in{\mathcal{G}} by (SA-2). Thus 𝒢′⊆𝒢{\mathcal{G}}^{\prime}\subseteq{\mathcal{G}} and we are done.

2.3 Probability measure

ℙ\mathbb{P} is known as the probability measure, which is a set-valued function ℙ:ℱ→[0,1]\mathbb{P}:{\mathcal{F}}\to[0,1] that assigns values to events consistently (known as probability axioms):

  • (PM-1)
    ​

    ℙ⁢(A)≥0\mathbb{P}(A)\geq 0 for any A∈ℱA\in{\mathcal{F}}.

  • (PM-2)
    ​

    ℙ⁢(∅︀)=0\mathbb{P}(\emptyset)=0 and ℙ⁢(Ω)=1\mathbb{P}(\Omega)=1.

  • (PM-3)
    ​

    (Countable additivity.) If An∈ℱA_{n}\in{\mathcal{F}} are mutually disjoint, i.e. Aj∩Ak=∅︀A_{j}\cap A_{k}=\emptyset for any j≠kj\neq k, then ℙ⁢(⋃n=1∞An)=∑n=1∞ℙ⁢(An).\mathbb{P}\left(\bigcup_{n=1}^{\infty}A_{n}\right)=\sum_{n=1}^{\infty}\mathbb{% P}(A_{n}).

Exercise 2.8.

Show that the assumption ℙ⁢(∅︀)=0\mathbb{P}(\emptyset)=0 in (PM-2) is redundant.

Solution: since ∅︀,Ω∈ℱ\emptyset,\Omega\in{\mathcal{F}} are mutually disjoint, we see that 1=ℙ⁢(Ω)=ℙ⁢(Ω∪∅︀)=ℙ⁢(Ω)+ℙ⁢(∅︀)=1+ℙ⁢(∅︀)1=\mathbb{P}(\Omega)=\mathbb{P}(\Omega\cup\emptyset)=\mathbb{P}(\Omega)+% \mathbb{P}(\emptyset)=1+\mathbb{P}(\emptyset) by (PM-2) and (PM-3), from which we obtain ℙ⁢(∅︀)=0\mathbb{P}(\emptyset)=0. Alternatively, ℙ⁢(∅︀)=ℙ⁢(∅︀∪∅︀)=2⁢ℙ⁢(∅︀)\mathbb{P}(\emptyset)=\mathbb{P}(\emptyset\cup\emptyset)=2\mathbb{P}(\emptyset).

There are a few basic equalities/inequalities regarding probability measures. We will list few of them below and leave them as revision exercise.

Exercise 2.9.

Prove the following claims.

  • (1)
    ​

    Inclusion-exclusion principle: if A,B∈ℱA,B\in{\mathcal{F}}, then ℙ⁢(A∪B)=ℙ⁢(A)+ℙ⁢(B)−ℙ⁢(A∩B)\mathbb{P}(A\cup B)=\mathbb{P}(A)+\mathbb{P}(B)-\mathbb{P}(A\cap B). In partcular, if A⊆BA\subseteq B, then ℙ⁢(A)≤ℙ⁢(B)\mathbb{P}(A)\leq\mathbb{P}(B).

  • (2)
    ​

    Continuity of probability measure: suppose An∈ℱA_{n}\in{\mathcal{F}} is a monotonically increasing sequence of events, i.e.

    A1⊆A2⊆A3⊆⋯\displaystyle A_{1}\subseteq A_{2}\subseteq A_{3}\subseteq\cdots

    then ℙ⁢(⋃n=1∞An)=limn→∞ℙ⁢(An)\mathbb{P}\left(\bigcup_{n=1}^{\infty}A_{n}\right)=\lim_{n\to\infty}\mathbb{P}% (A_{n}). Hence, establish an analogous claim for monotonically decreasing sequence of events, i.e.

    A1⊇A2⊇A3⊇⋯⇒ℙ⁢(⋂n=1∞An)=limn→∞ℙ⁢(An).\displaystyle A_{1}\supseteq A_{2}\supseteq A_{3}\supseteq\cdots\qquad% \Rightarrow\qquad\mathbb{P}\left(\bigcap_{n=1}^{\infty}A_{n}\right)=\lim_{n\to% \infty}\mathbb{P}(A_{n}).
  • (3)
    ​

    Union bound: if An∈ℱA_{n}\in{\mathcal{F}} is a sequence of events, then ℙ⁢(⋃n=1∞An)≤∑n=1∞ℙ⁢(An)\mathbb{P}\left(\bigcup_{n=1}^{\infty}A_{n}\right)\leq\sum_{n=1}^{\infty}% \mathbb{P}(A_{n}).

Solution: we only sketch some arguments here but details can be found in your first year probablity notes.

  • (2)
    ​

    Note that the analogous result for monotonically decreasing sequence of events follows directly using De Morgan’s law:

    ℙ⁢(⋂n=1∞An)=1−ℙ⁢(⋃n=1∞Anc)=1−limn→∞ℙ⁢(Anc)=limn→∞ℙ⁢(An).\displaystyle\mathbb{P}\left(\bigcap_{n=1}^{\infty}A_{n}\right)=1-\mathbb{P}% \left(\bigcup_{n=1}^{\infty}A_{n}^{c}\right)=1-\lim_{n\to\infty}\mathbb{P}% \left(A_{n}^{c}\right)=\lim_{n\to\infty}\mathbb{P}(A_{n}).
  • (3)
    ​

    The union bound follows easily from countable additivity. If we write Bn=⋃k=1nAkB_{n}=\bigcup_{k=1}^{n}A_{k} and Cn:=An∖BnC_{n}:=A_{n}\setminus B_{n} (with the convention that C1:=A1C_{1}:=A_{1}), then Cn⊆AnC_{n}\subseteq A_{n} and are mutually disjoint and ⋃n=1∞Cn=⋃n=1∞An\bigcup_{n=1}^{\infty}C_{n}=\bigcup_{n=1}^{\infty}A_{n}. Hence

    ℙ⁢(⋃n=1∞An)=ℙ⁢(⋃n=1∞Cn)=∑n=1∞ℙ⁢(Cn)≤∑n=1∞ℙ⁢(An).\displaystyle\mathbb{P}\left(\bigcup_{n=1}^{\infty}A_{n}\right)=\mathbb{P}% \left(\bigcup_{n=1}^{\infty}C_{n}\right)=\sum_{n=1}^{\infty}\mathbb{P}(C_{n})% \leq\sum_{n=1}^{\infty}\mathbb{P}(A_{n}).
