Q2. The linearity of conditional expectation says: if \(X, Y\)$ are two integrable random variables on a probability space \((\Omega, \mathcal{F}, \mathbb{P})\) and \(\mathcal{G} \) is a sub \(\sigma\)-algebra, then \( \mathbb{E}[X + Y | \mathcal{G}] = \mathbb{E}[X | \mathcal{G}] + \mathbb{E}[Y | \mathcal{G}]\) a.s. The following is a proof of the claim:

  1. Since \(X, Y\) are integrable, so is \(X+Y\).
  2. Let \(U = \mathbb{E}[X | \mathcal{G}]\) and \(V = \mathbb{E}[Y | \mathcal{G}]\). It suffices to show that \(\mathbb{E}[(X+Y); A] = \mathbb{E}[(U+V); A]\) for all \(A \in \mathcal{G}\).
  3. But \(\mathbb{E}[(X+Y); A] = \mathbb{E}[X; A] + \mathbb{E}[Y; A] \)
  4. \( = \mathbb{E}[U; A] + \mathbb{E}[V; A] \)
  5. \( = \mathbb{E}[(U+V); A] \).
Choose the correct keywords below that describe each step in the proof.
  • Triangle inequality, independence, monotonicity, lienarity of usual expectation, definition, take out what is known, tower property.

Q3. Let \(X_1, X_2\) be two random variables on the same probability space satisfying \(\mathbb{E}[\exp(t (|X_1| + |X_2|))] < \infty\) for any fixed \(t > 0\). Which of the following statement is true?

  1. \(X_1^n, X_2^n\) are both integrable for any fixed integer \(n\).
  2. If \(X_1, X_2\) are independent of each other, then \(\mathrm{Cov}(X_1, X_2) = 0\).
  3. If \(\mathrm{Cov}(X_1, X_2) = 0\), then \(X_1, X_2\) are independent of each other.
  4. If \(\mathbb{E}[\exp(t (X_1 + X_2))] = \mathbb{E}[\exp(t X_1))] \mathbb{E}[\exp(t X_2)]\) for all \(t > 0\), then \(X_1, X_2\) are independent of each other.
